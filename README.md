# 体重管理アプリ

スマートフォン最適化の体重管理 Web アプリです。HTML / CSS / JS のみで動作し、ビルド不要です。

## 起動方法

`index.html` をブラウザで開くだけで使用できます。

```
open index.html   # macOS
```

## ファイル構成

```
my-weight-app/
├── index.html   # メイン HTML
├── style.css    # スタイルシート（スマホ最適化）
├── app.js       # アプリケーションロジック
└── README.md    # このファイル
```

## プロフィール初期値

| 項目 | 値 |
|------|----|
| 年齢 | 45 歳 |
| 性別 | 男性 |
| 身長 | 170 cm |
| 現在体重 | 69 kg |
| 目標体重 | 66 kg |

## 機能一覧

### プロフィール管理
- 年齢・性別・身長・現在体重・目標体重を設定・保存

### 毎日の記録
- 日付・体重・歩数・走った距離を記録
- 同じ日付の記録は上書き保存

### 計算機能

| 計算 | 計算式 |
|------|--------|
| 基礎代謝（男性） | 88.362 + 13.397×体重 + 4.799×身長 − 5.677×年齢 |
| 基礎代謝（女性） | 447.593 + 9.247×体重 + 3.098×身長 − 4.330×年齢 |
| 歩行消費カロリー | 歩数 ÷ 1,000 × 35 kcal |
| 走行消費カロリー | 距離(km) × 60 kcal |

### 食事提案
- 2,100 kcal 目安の朝食・昼食・夕食・おやつを提案
- 納豆・豆腐・鶏胸肉・鮭・ブロッコリーなど高タンパク低カロリー食材を紹介

### 体重推移グラフ
- 直近 30 件のデータを折れ線グラフで表示
- 目標体重ラインを赤破線で表示
- 直近 14 件の記録を表形式で一覧表示

## データ保存

localStorage を使用してブラウザ内に保存します。ブラウザのデータを削除するとリセットされます。

## 動作環境

- Chrome / Safari / Firefox（最新版推奨）
- スマートフォン・タブレット・PC で動作
